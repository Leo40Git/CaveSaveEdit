General Strings (prefix "General."):
- (String) Name: Name of the mod these defines are associated with.
- (String) Author: Name(s) of the author(s) of the defines.
- (String) None: Generic "none" string.
           Used if there is no specific "none" string for the current type.
Special Support (prefix "Special."):
- (Bool) MimHack: Adds support for the <MIM hack. Prevents the used flags
         from being edited in the flag UI and allows modifying the current costume ID. 
- (Bool) VarHack: Adds support for the <VAR (TSC+) hack. Prevents the used flags
         from being edited in the flag UI and allows modifying the variables.
- (Bool) PhysVarHack: Adds support for the <PHY (TSC+ PHY addon) hack. Prevents the used flags
         from being edited in the flag UI and allows modifying the physics variables.
